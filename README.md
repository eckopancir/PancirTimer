# PancirTimer v4.0

Легковесное приложение для Windows 10/11 для управления напоминаниями (лекарства, тренировки, сериалы).

## Особенности
- **Современный UI**: Использован `CustomTkinter` с бело-оранжевой цветовой схемой.
- **Уведомления**: Системные уведомления Windows (Toast) со звуком.
- **Работа в фоне**: Приложение сворачивается в системный трей при закрытии окна.
- **Автозагрузка**: Возможность включить запуск вместе с Windows прямо из интерфейса.
- **Хранение данных**: Все задачи сохраняются в файл `tasks.json`.

## Как запустить
1. Убедитесь, что установлен Python 3.
2. Установите зависимости (уже сделано в текущем окружении):
   ```bash
   pip install customtkinter pystray winotify pillow
   ```
3. Запустите приложение:
   ```bash
   python main.py
   ```

## Как пользоваться
- Введите название (например, "Витамин C") и время в формате `HH:MM`.
- Нажмите "**Добавить**".
- Чтобы приложение работало в фоне, просто закройте окно — оно останется в трее (возле часов).
- Чтобы полностью выйти, кликните правой кнопкой мыши по иконке в трее и выберите "**Выход**".
- Чтобы снова открыть окно из трея, выберите "**Открыть**".
